Frontend

Definición:

Se refiere a la parte de una aplicación web o movil con la que los usuarios interactuan directamente.
Es el componente visible y accesble que incluye la interfaz de usuario (UI)
asi como todas las interacciones que los usuarios experimenta en la pagina.

Fundamentos:



Esta basado en varias tecnologias y principios clave que
permiten que los sitios web sean interactivos,rapidosy accesibles.

1.HTML (HyperText Markup Language)
2.CSS (Cascading style Sheets)
3.JavaScript
